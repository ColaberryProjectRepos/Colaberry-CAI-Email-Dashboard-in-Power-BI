2. Data Model 2


Data Model for CAI Dashboard

The below data model was created in Power BI for CAI dashboard. The four tables Email Campaigns, Email Groups, Email Severity and Email History were retrieved by connecting to SQL Server. Rest of the tables were created in Power BI.



Email Campaigns

The Email Campaigns table contains all the data related to the different campaigns such as Login Rate Below 70 %, Activity Rate Dropping, Last Week Intern Score - Bad, Failed SQL Server Agent Job, Email your mentees, Earn $50-$200 for Referrals to name a few.





Email Groups

The Email Groups table contains all the information related to different groups in Colaberry such as IPBC, JRP, Internship Program, Mentorship Management, Alumni to name a few.




Email Severity

As the name suggests, the Email Severity table contains information about Severity Name (such as Urgent, High, Medium and Low), Severity Order and the Email Template to use.





Email History

The Email History table is the main FACT table containing the history of all the emails sent out by CAI. It contains information such as Email ID, Sent Date, Email Opens, Email Clicks etc.



Date Table

The Date Table was created for Time dimension. It contains all the dates from start of Calendar year 2019 to current date.



Date Period

The Date Period table was created for the Time Slicer for different periods such as Current Week, Last Week, 2 Weeks Ago, 3 Weeks Ago, Rolling 4 Weeks etc. For Project documentation purposes, 3-Jun-2019 was taken as-of date to be able to show data for all time periods on the dashboard. The MAX(SentDate) should be replaced with Today() for actual purposes.



Email-Name Cross Reference 

Lastly Email-Name Cross Reference table was created to replace all instances of Email with Names. Its a static table.
